HOW TO OPEN THE WEBSITE LOCALLY

1. Right-click the ZIP file and choose Extract / Unzip.
2. Open the extracted folder: lilys-kvalitet-stad-website
3. Double-click index.html.
4. Do NOT open the HTML while it is still inside the ZIP preview.
5. If images still do not show, open the website through a local server:
   - In VS Code, install Live Server
   - Right-click index.html
   - Click “Open with Live Server”

This package uses relative image paths, so images should work after extraction.
